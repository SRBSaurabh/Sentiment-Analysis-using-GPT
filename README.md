# Movie Reviews Sentiment Classification from Rotten Tomatoes

This project performs sentiment classification on movie reviews from the Rotten Tomatoes dataset using NLP techniques.

## What This Project Does
- Loads the Rotten Tomatoes movie reviews dataset from Hugging Face
- Applies NLP models to classify reviews as positive or negative
- Uses transformers and sentence embeddings for improved accuracy

## Installation

```bash
pip install datasets transformers sentence-transformers
```

## Usage

Run the main script to train and evaluate the sentiment classification model:

```bash
python main.py
```

## Files
- `main.py` - The main script to run the classification pipeline
- `README.md` - Documentation for setup and usage

## Requirements
- Python 3.7+
- Hugging Face Datasets
- Transformers
- Sentence Transformers
